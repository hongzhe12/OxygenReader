# Oxygen Reader | 氧气阅读器

**像氧气一样：轻盈、透明、不可或缺，却又难以察觉。**

## 📖 简介

**Oxygen Reader** 是一款专为 Windows 10/11 用户打造的**极简主义透明阅读器**。

它打破了传统阅读软件的"边框"束缚，将文字直接悬浮于你的桌面壁纸、Excel 表格或代码编辑器之上。

## ✨ 核心特性

### 👻 极致隐身
- **全透明悬浮**：无边框、无背景，文字直接浮在屏幕上。
- **一键消失**：鼠标左键点击屏幕任意空白处，或按下 `Ctrl+Shift+Q`，文字瞬间隐形。
- **托盘驻留**：控制台最小化至系统托盘（右下角），任务栏不留痕迹。

### 🛡️ 变色龙伪装
- **深度伪装**：支持自定义窗口标题及图标。
- **环境融合**：字体大小、颜色、透明度完全可调。

### 🎯 聚焦引导
- **启动高亮**：开始阅读时，文字自动居中并**红字高亮**。
- **自动隐退**：翻页后立即恢复隐蔽色，回归潜行模式。

### 📚 全能解析 & 智能记忆
- **格式通吃**：原生支持 **PDF, EPUB, MOBI, AZW3, TXT** 等主流电子书格式。
- **数据持久化**：阅读进度（精确到行）、书架记录保存在系统 `%APPDATA%` 目录。

## 🛠️ 使用

### 源码运行

1. **安装依赖**
   ```bash
   pip install PyQt5 pymupdf keyboard requests pynput pyinstaller

2. **运行代码**
   ```bash
   python main.py

3. **打包成 EXE**
   ```bash
pyinstaller --onefile --noconsole --icon=logo.ico --add-data "logo.ico;." main.py

| 功能 | 默认按键 | 说明 |
